Tipos de Unión

Problema

A veces una variable puede ser de varios tipos (ej: un ID que es número o texto). JS no nos deja definir esta restricción.

Solución

TypeScript permite usar el operador '|' para definir uniones de tipos.

Analogía Visual

"Como un agente doble. Puede ser civil o espía, pero nada más. Tú decides qué roles están permitidos."

let id: string | number;
id = 123; // OK
id = 'abc'; // OK
id = true; // ERROR: Boolean no está permitido

Pregunta de Entrevista

Q: ¿Qué es un Union Type?

A: Permite que un valor sea uno de varios tipos posibles, aumentando la flexibilidad sin perder la seguridad.
