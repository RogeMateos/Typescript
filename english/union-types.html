Union Types

Problem

Sometimes a variable can be of several types (e.g., an ID that is number or text). JavaScript doesn't let us define this constraint.

Solution

TypeScript allows using the '|' operator to define unions of types.

Visual Analogy

"Like a double agent. Can be civilian or spy, but nothing else. You decide which roles are allowed."

let id: string | number;
id = 123; // OK
id = 'abc'; // OK
id = true; // ERROR: Boolean is not allowed

Interview Question

Q: What is a Union Type?

A: It allows a value to be one of several possible types, increasing flexibility without losing safety.
