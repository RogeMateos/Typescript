Enums: Valores Fijos

Problema

Usar strings mágicos como 'admin' o 'user' genera errores tipográficos (ej: 'admmin') difíciles de encontrar.

Solución

Los Enums crean un conjunto de constantes con nombre, facilitando la lectura y evitando errores.

Analogía Visual

"Como un semáforo. Solo tiene tres estados posibles: Rojo, Amarillo o Verde. No hay 'Azul'."

enum Role {
  Admin,
  User,
  Guest
}
const myRole: Role = Role.Admin;

Pregunta de Entrevista

Q: ¿Cuándo usar un Enum?

A: Cuando tienes un conjunto limitado de valores fijos y quieres evitar el uso de strings sueltos o números mágicos.
